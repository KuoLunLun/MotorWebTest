使用工具分享

Visual Studio Code

Visual Studio Code（簡稱VS Code）是一個由微軟開發的，同時支援Windows、Linux、和macOS系統且開放原始碼的程式碼編輯器，它支援測試，並內建了Git 版本控制功能，同時也具有開發環境功能，例如程式碼補全（類似於 IntelliSense）、程式碼片段、和程式碼重構等，該編輯器支援用戶個性化組態，例如改變主題顏色、鍵盤捷徑等各種屬性和參數，還在編輯器中內建了擴充程式管理的功能。

相關連結：
Visual Studio Code

Adobe XD

Adobe Experience Design(Adobe XD) 為 Adobe Inc 發布的向量繪圖軟體，用來設計網頁及行動應用程式的使用者經驗。能在macOS、Windows、iOS和Android使用。 XD擁有向量繪圖設計及網站線框圖(Wireframe)設計的功能，能創造簡單的點擊互動原型(Prototype)。

相關連結：
Adobe XD

Adobe Photoshop

Adobe Photoshop，簡稱「PS」，是一個由Adobe開發和發行的影像處理軟體。該軟體發布在Windows和Mac OS上。

相關連結：
Adobe Photoshop

Adobe Illustrator

Adobe Illustrator，簡稱「AI」，是Adobe系統公司推出的基於向量的圖形製作軟體。最初是1986年為蘋果公司麥金塔電腦設計開發的，1987年1月發布，在此之前它只是Adobe內部的字型開發和PostScript編輯軟體。

相關連結：
Adobe Illustrator

Bootstrap 4

Bootstrap是一組用於網站和網路應用程式開發的開源前端（所謂「前端」，指的是展現給終端使用者的介面。與之對應的「後端」是在伺服器上面執行的程式碼）框架，包括HTML、CSS及JavaScript的框架，提供字體排印、表單、按鈕、導航及其他各種元件及Javascript擴充套件，旨在使動態網頁和Web應用的開發更加容易。

相關連結：
Bootstrap

Codepen

Codepen 是近幾年很受歡迎的線上編輯器。與同為線上編輯器的 jsFiddle 相比，介面設計美觀許多，點進網頁就可以看到許多厲害的作品。Codepen 的 EXPLORE 做得很好，從進入首頁就可以看到各種作品，你也可以透過輸入 tag 搜尋你想看的內容，且分類也算是細緻，從 Pen 到 Project 都有，你可以依照個人想搜尋的專案大小來分別尋找更符合自己想看到的內容，甚至可以加入收藏，是兼具美觀與社群的超棒編輯器。

相關連結：
Codepen.io

Swiper

Swiper - 是免費且最現代化的移動觸摸滑塊，具有硬件加速過渡和驚人的原生行為。它旨在用於移動網站，移動網絡應用程序和移動本機/混合應用程序。主要針對iOS設計，但也適用於最新的Android，Windows Phone 8和現代桌面瀏覽器 Swiper與所有平台不兼容，它是一個現代觸摸滑塊，僅專注於現代應用程序/平台，以帶來最佳體驗和簡單性。
